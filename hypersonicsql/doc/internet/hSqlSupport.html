Support

Public Survey

Please help us to improve Hypersonic SQL and tell us which features are most important to you. The public survey is hosted by SourceForge. Unfortunately, you have to be logged in SourceForge to participate in surveys, but it is quite easy to log in as a new user. Please take this time. Public Survey.

Support Request

As a first step, please read the How To / FAQ (Frequently Asked Questions).

If you still need support, a user mailing list has been set up. You can:

Send a mail to hsql-user@lists.sourceforge.net
Subscribe to this mailing list
Read the archives.

Register a Bugs / Feature Requests

If you have to report a bug in Hypersonic SQL please use the bug reporting system at SourceForge. If possible include also the code that caused the error. The error(s) can then be fixed much easier and faster. Use this form also to post a new feature request: Bugs / Feature Requests

News

If you would like to receive news about Hypersonic SQL by email, send a mail to thomasm2@bluewin.ch with subject 'Hypersonic SQL news'. News mail will be sent at most once every month. Guaranteed no spam, and your mail address will not be used for something else.

Development

There are a lot of future plans for this project. For example: extended SQL syntax and functionality, and an improved client/server version. If you would like to develop additional code for this project, please send a mail to: thomasm2@bluewin.ch.
